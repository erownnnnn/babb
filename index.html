EvLaunch Attendance System

RFID Attendance System • JavaFX • MySQL

EvLaunch is an RFID-based attendance management system designed to automate and streamline student attendance tracking. The system uses RFID technology to scan student IDs and instantly record attendance into a MySQL database. Developed with Java for the backend and JavaFX for the user interface, the application provides a modern desktop experience with multiple features such as attendance monitoring, student management, real-time logs, and database integration. The project highlights the combination of hardware and software technologies to create an efficient and reliable attendance solution.

Feline

Machine Learning • Machine Vision • APK & Web App

Feline is a cat breed identifier application that combines machine learning and machine vision technologies to recognize cat breeds in real time. Available both as an offline-capable APK application and a responsive website, the system can detect and classify five cat breeds using a Teachable Machine-trained model. Built using HTML, CSS, and JavaScript, the application supports live camera detection and image uploads, providing an accessible AI-powered experience across devices.

Pixel Studio

Image Processing • Pure Python

Pixel Studio is a custom-built image processing application developed entirely in pure Python without relying on external image processing libraries. The system features 15 different processing tools and filters that generate unique outputs from uploaded images. Designed to strengthen core programming and algorithmic skills, the project showcases manual implementation of image manipulation techniques such as filtering, transformations, and pixel-level operations.

Starship Wars

Machine Vision • Python Game • OpenCV

Starship Wars is an interactive space shooting game powered by machine vision technology. The game uses real-time hand gesture recognition through a webcam, allowing players to control the spaceship without using a keyboard or mouse. Built with Python using OpenCV and MediaPipe, the project demonstrates the integration of computer vision, gesture tracking, and real-time gameplay mechanics in a fun and immersive experience.

UA (Unta Ako)

Embedded Systems • Arduino Nano • PID Control

UA (Unta Ako) is an automated ball balancing system designed to keep a ball centered on a platform using real-time feedback control. The project utilizes an Arduino Nano, servo motors, and an ultrasonic sensor to detect the ball’s position and continuously adjust the platform angle. To achieve stable and accurate balancing, the system implements PID control and Median Filter algorithms for smoother sensor readings and precise motion correction. This project demonstrates the application of embedded systems, control systems, and robotics in creating an intelligent balancing mechanism.

Smart Door System

IoT • ESP32 • RFID Security System

The Smart Door System is an intelligent security and access control project powered by ESP32 technology. The system uses an ultrasonic sensor to detect objects or people approaching the door and an RFID scanner to verify whether the scanned ID is registered and authorized for access. Once authentication is successful, a servo motor automatically unlocks and opens the door. Designed for automation and security applications, the project demonstrates the integration of IoT, embedded systems, and sensor-based authentication for smart access management.

Rootfinder using Python

Numerical Methods • Python App

This project is a powerful root-finding application built with Python that visualizes and solves equations using six numerical methods: Graphical, Incremental Search, Bisection, Regula Falsi, Newton-Raphson, and Secant. It supports dynamic input, multi-root detection, tabulated iterations, and interactive plots—making it an essential tool for students and professionals in computational mathematics.
